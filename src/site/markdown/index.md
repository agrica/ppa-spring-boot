ppa-spring-boot
=========================


## Description
Pom parent utilisé pour les projet Spring Boot.

## Usage
Dans le POM maven du projet héritez du PPA Agrica
```xml
  <parent>
    <groupId>io.github.agrica</groupId>
      <artifactId>ppa-spring-boot</artifactId>
      <version>2.Y.Z</version>
   </parent>
```
